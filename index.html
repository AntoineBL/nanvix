What is Nanvix?

Nanvix is a free and open source operating system created by Pedro H. Penna for instructional purposes. It is designed to be simple and small, but also modern and fully featured. The project started in January of 2011 and it stills active and growing.

The Nanvix operating system has three main components. The Nanvix Kernel targets the x86-based PC platform, has a monolithic and non-preemptive architecture, and follows Posix standards. The Nanvix C Library is a platform independent implementation of the standard C that has been ported and adapted from the Free BSD 4.0 Operating System. Finally, the Nanvix Utilities consists in a collection of Unix-like utilities for the Nanvix platform.
