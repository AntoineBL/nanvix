What is Nanvix?

Nanvix is a Unix like operating system created by Pedro H. Penna to address the demands of emerging manycore processors. It targets cluster-based architectures that rely on distributed and shared memory, and it was designed from scratch to deliver cutting edge performance, while enabling backward compatibility with existing software.

The Nanvix operating system has three main components:

  • The Nanvix Kernel provides an abstraction of the underlying hardware. It presents a monolithic architecture, runs in privileged and non interruptible mode, and features time sharing, multiprogramming, multithreading and a virtual file system.

  • The Nanvix C Library consists in a platform-independent implementation of the C Library. It has been derived from Red Hat's Newlib and it is fully compatible with the ISO standards.

  • The Nanvix Utilities is a set of bare bones user programs for administrating, monitoring and benchmarking the operating system.
