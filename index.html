What is Nanvix?

Nanvix is a free and open source operating system created by Pedro H. Penna for instructional purposes. It is designed to be simple and small, but also modern and fully-featured. The project started in January 2011 and it stills active and growing.

The Nanvix operating system has three main components: the Nanvix Kernel targets the x86-based PC platform, has a monolithic and non-preemptive architecture, and follows Posix standards; the Nanvix C Library is a platform- independent implementation of the standard C library that has been ported and adapted from the Free BSD 4.0 Operating System; finally, the Nanvix Utilities consist in a collection of Unix-like utilities for Nanvix.
