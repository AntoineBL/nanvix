How to install Nanvix?

Nanvix is an operating system that targets the x86 architecture, requiring a IBM PC compatible environment with 16 MB of RAM and a floppy or CD-ROM drive. Therefore, you can run Nanvix either in a real PC or in a virtual machine. In both cases, you can download and use a live system image to run Nanvix without installing it on the host platform.

How to get involved?

Nanvix is an open source project, which means that you can download its source code and modify it under the project's license. The Nanvix documentation can help you understand the project's architecture, though it is not mature yet. If you are looking for someway to start coding faster, you can follow a quick-start tutorial and discover Nanvix's source code by your own.
