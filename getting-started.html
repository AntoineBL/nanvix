How to install Nanvix?

Nanvix is an operating system that targets the x86-based PC platform. It requires 32 MB of RAM, a floppy or CD-ROM drive, and a hard disk drive with 64 MB of free space available.

You may run Nanvix either in a real PC or in a virtual machine. Nevertheless, in both cases, if you do not want to install Nanvix you can try a live system image.

How to get involved?

Nanvix is an open source project, which means that you can download the source code, modify it under the project's license, and submit your changes.

If you fall into any troubles, you can either refer to the Nanvix documentation or contact one of the contributors.
