Road Map for Future Releases

The latest release version of is Nanvix is 1.2.

However, the development team is working hard on the following issues to deliver the next major version as soon as possible:

  • Add QEMU Support
  • Add VBE Support
  • Enhance the Process Scheduler
  • Enhance Virtual Memory Support
  • Enhance the Paging Replacement Algorithm
  • Add Multithreading Support
